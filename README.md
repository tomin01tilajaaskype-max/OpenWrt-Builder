# OpenRouterLab

OpenRouterLab is a GitHub Actions based build helper for custom OpenWrt firmware experiments.

Current first target:

- ZBTLink ZBT-Z8803BE / MT7988A / Wi-Fi 7 / Qualcomm X62

This repository is designed so the build happens in GitHub Actions. You do not need to build OpenWrt locally.

## Quick start

1. Create a new GitHub repository.
2. Upload this project content.
3. Open **Actions**.
4. Run **Build ZBT Z8803BE** manually.
5. Download the firmware from the workflow artifacts.

## Important

This is an experimental build base. Keep a recovery method available before flashing custom firmware.
